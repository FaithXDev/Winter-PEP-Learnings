# DSA Solutions - LeetCode

Welcome to my repository of Data Structures and Algorithms solutions for LeetCode problems. This collection is implemented in **Java** and focuses on clean, efficient, and well-documented code.

## üìÇ Project Structure

Each file in this repository represents a solution to a specific LeetCode problem. The files are self-contained and include:
- **Solution Class**: The core logic to solve the problem.
- **Main Method**: Hardcoded test cases to verify the solution locally.
- **Detailed Explanation**: A comment block at the end of each file explaining the approach, algorithm, and complexity analysis (Time & Space).

## üöÄ How to Run

You can run any solution file individually using a Java compiler or an IDE (like VS Code, IntelliJ IDEA, or Eclipse).

### Using Command Line
1. **Compile**:
   ```bash
   javac FileName.java
   ```
2. **Run**:
   ```bash
   java FileName
   ```

   *Example*:
   ```bash
   javac Two_Sum.java
   java Two_Sum
   ```
